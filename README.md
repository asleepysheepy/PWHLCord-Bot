## Adding PWHLcord Bot To Your Server

If you want to have PWHLcord Bot on your server, you can either invite the public
PWHLcord Bot account or set up your own Discord bot with the PWHLcord Bot code.

### Inviting the public PWHLcord Bot to your server

If you want to invite the public PWHLcord Bot account to your server, you can use
this invite link:
[PWHLcord Bot invite link](https://discord.com/oauth2/authorize?client_id=1448154918883164232)

## Frameworks, Libraries, and Dependencies

PWHLcord Bot is built with the following:

- [Biome](https://biomejs.dev/) - Formatting and linting
- [BullMQ](https://bullmq.io/) - Event Queue Management
- [Discord.js](https://discord.js.org) - Discord API for Node.js
- [Docker](https://www.docker.com/) - Containerization for easier development and
  deployment
- [Drizzle](https://orm.drizzle.team/) - ORM for interacting with the database
- [node-cron](https://nodecron.com/) - Job scheduling
- [Pino](https://getpino.io) - Logging and log management
- [PostgreSQL](https://www.postgresql.org/) - Persistent storage
- [Zod](https://zod.dev/) - Schema validation
